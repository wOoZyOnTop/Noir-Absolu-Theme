# Noir Absolu / Absolute Black

One true-black AMOLED Discord theme, with any accent color you want. No CSS editing is required.

Un seul thème Discord AMOLED en noir absolu, avec la couleur d’accent de votre choix. Aucune modification CSS n’est nécessaire.

## Color configurator / Configurateur de couleur

### [Choose any color and download your theme / Choisir une couleur et télécharger votre thème](https://woozyontop.github.io/Noir-Absolu-Theme/)

The configurator creates the personalized `.theme.css` file directly in your browser. Nothing is uploaded, and the colored dots are only quick presets—not separate themes.

Le configurateur crée le fichier `.theme.css` personnalisé directement dans votre navigateur. Aucune donnée n’est envoyée et les pastilles sont seulement des raccourcis, pas des thèmes distincts.

[Download the default purple theme / Télécharger le thème violet par défaut](https://github.com/wOoZyOnTop/Noir-Absolu-Theme/releases/latest/download/Noir-Absolu.theme.css)

## Installation

### BetterDiscord

1. Open the [color configurator](https://woozyontop.github.io/Noir-Absolu-Theme/).
2. Choose any color and download `Noir-Absolu-Personnalise.theme.css`.
3. Open `Discord Settings > BetterDiscord > Themes > Open Themes Folder`.
4. Place the downloaded file in that folder and enable it.

### Vencord — local file / fichier local

1. Ouvrez le [configurateur de couleur](https://woozyontop.github.io/Noir-Absolu-Theme/).
2. Choisissez une couleur et téléchargez `Noir-Absolu-Personnalise.theme.css`.
3. Ouvrez `Paramètres Discord > Vencord > Themes > Open Themes Folder`.
4. Placez le fichier dans ce dossier et activez-le.

### Vencord — built-in color picker / sélecteur intégré

You can also use the canonical theme with Vencord’s built-in `ClientTheme` plugin. Enable `ClientTheme`, choose a color in its visual picker, then use this single Online Themes URL:

Vous pouvez aussi utiliser le thème canonique avec le plugin `ClientTheme` intégré à Vencord. Activez `ClientTheme`, choisissez la couleur dans son sélecteur visuel, puis utilisez cette URL unique dans Online Themes :

```text
https://woozyontop.github.io/Noir-Absolu-Theme/Noir-Absolu.theme.css
```

## Features / Fonctionnalités

- True OLED black (`#000000`) across Discord’s main surfaces
- Any accent color through the visual web configurator
- Automatic browser-language detection (FR, EN, DE, ES, IT, PT, NL, PL)
- Native support for Vencord’s `ClientTheme` color picker
- Compact scrollbars and a completely black voice/user panel area
- Clear styling for links, mentions, replies, selections, buttons, borders, and focus states
- Compatible with BetterDiscord and Vencord
- One shared theme core, so future fixes apply automatically to personalized files

## Important

Enable either the default theme or your personalized file, not both at the same time.

Activez soit le thème par défaut, soit votre fichier personnalisé, mais pas les deux en même temps.

Only `Noir-Absolu.theme.css` is the canonical theme intended for publication. Personalized downloads are local configurations of this same theme.

Seul `Noir-Absolu.theme.css` est le thème canonique destiné à être publié. Les téléchargements personnalisés sont des configurations locales de ce même thème.
